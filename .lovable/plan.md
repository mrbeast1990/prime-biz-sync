

# تصدير واستيراد المنتجات مع تعديل البيانات قبل الحفظ

## الملخص
ربط أزرار التصدير والاستيراد الموجودة فعلياً في صفحة المنتجات بوظائف حقيقية، مع إضافة نافذة مراجعة وتعديل البيانات المستوردة قبل تأكيد الإضافة.

## التعديلات المطلوبة

### 1. ملف جديد: `src/components/products/ImportPreviewDialog.tsx`
نافذة Dialog كبيرة تعرض البيانات المستوردة من CSV في جدول قابل للتعديل:
- جدول يعرض الأعمدة: الكود، الاسم التجاري، الاسم العلمي، التعبئة، الوحدات، سعر التكلفة، سعر البيع، المخزون، الحد الأدنى، التصنيف، خاضع للصلاحية
- كل خلية قابلة للتعديل مباشرة (inline editing) عبر inputs
- إمكانية حذف صف من القائمة قبل التأكيد (زر حذف لكل صف)
- عرض عدد الأصناف الجاهزة للإضافة
- زر "تأكيد الإضافة" يرسل كل المنتجات دفعة واحدة
- زر "إلغاء" يغلق النافذة بدون حفظ
- تنبيهات للأخطاء (مثل حقول مطلوبة فارغة)

### 2. تحديث: `src/utils/exportUtils.ts`
- إضافة دالة `parseCSV(text: string): Record<string, string>[]` لتحليل ملف CSV مع دعم UTF-8 + BOM
- إضافة دالة `mapCSVToProducts(rows)` لتحويل أعمدة CSV (عربية أو إنجليزية) إلى بنية `Partial<Product>`
- تحديث `exportToCSV` لتصدير المنتجات بأسماء أعمدة عربية واضحة

### 3. تحديث: `src/pages/Products.tsx`
- ربط زر "تصدير" بدالة تصدير CSV للمنتجات الحالية (مع أسماء أعمدة عربية)
- ربط زر "استيراد" بفتح file input مخفي يقبل `.csv`
- عند اختيار ملف: قراءته → تحليله → عرض `ImportPreviewDialog`
- عند تأكيد الاستيراد: إرسال المنتجات واحداً واحداً أو دفعة عبر `useCreateProduct`

### التفاصيل التقنية

**تصدير CSV:**
```text
الأعمدة المصدرة:
الكود | الاسم التجاري | الاسم العلمي | التصنيف | التعبئة | الوحدات | سعر التكلفة | سعر البيع | المخزون | الحد الأدنى | خاضع للصلاحية
```

**استيراد CSV:**
- يقبل نفس التنسيق المصدر (أعمدة عربية) + أعمدة إنجليزية كبديل
- `FileReader` لقراءة الملف كـ text مع encoding UTF-8
- عرض النتائج في Dialog قابل للتعديل

**نافذة المراجعة:**
- Dialog بعرض `max-w-6xl` مع scroll area للجدول
- كل خلية فيها input صغير للتعديل المباشر
- صف الأخطاء يُلون بالأحمر (مثل اسم تجاري فارغ)
- شريط سفلي يعرض: عدد الأصناف + زر تأكيد + زر إلغاء
